sabse pehle I created the project with the command
npm create vite@latest

fir maie usme tailwind aur baaki saari dependencies install kari using the command
npm i

fir maine development server chalaya
using npm run dev

fir maine tailwind ka setup kiya
vite.config wali file me tailwind ko add kiya
aur index.css me tailwind ko import kiya

fir maine sabse pehle header banaya
jisme maine title likha aur search ki ui banayi
maine ye sab components folder bana ke usme alag alag component bana ke add kara hai

fir maine fakestore ki api se products fetch kiye hai

fir maine unko log karke check kiya ki mere paas me sahi response hai ya nahi
sahi response aane ke baad me maine card component banaya

aur maine sabse pehle saare products us card component ko diye

Card components ke andar maine product ki image, title, price aur view details ka button banaya hai

fir maine search functionality ko banaya jisme maine bas fetched products ko filter kiya hai
